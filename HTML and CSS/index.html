Best Practise - HTML,CSS



1-HTML


HTML strukturasi
Html,Head,Body - tartibi to'g'ri bo'lishi
Semantik qodilariga rioya qilish - Header,Nav,Main,Section,Footer
Toza code yozish
Kodni tartibli joylashtirish
Teglarni yopishni unutmang

2-CSS


CSS asosiy qodilari
Class va ID dan to'g'ri foydalanish
Inline style dam qochish alohida CSS fayl ishlatish best practise qoidasiga to'g'ri keladi
Codeni tartib bilan yozish, masalan birinchi style berish va detallarga
Moslashuvchanlik - (responsivness)
px o'rniga %, vh, vw, rem, em lardan foydalanish
flex yoki grid dan foydalanish
Accessibility - saytimizni qulayligi foydalunchiga
img tagida ichida alt yozish
Button va inputlarni to'g'ri ishlatish

Nimalardan qochish kerak

  1. (br br br) bilan joy ajratish o'rniga margin yoki padding dan foydalanish
  2. ortiqcha div ishlatish
  3. h1 ni sahifa faqat bir marta ishlatish
  4. !important ni ko'p ishlatishdan qochish
